基于xillinx的FPGA开发的自动音乐播放器

使用spartan xc3s50an的开发板，基于win10运行的ise14.7进行开发

实现功能：通过一个单独按键，每按一次自动切换到下一个音乐播放，共有3首音乐

原理：通过晶振分频过后获得一个基本方波信号，再通过两个分频电路，第一个进行12阶音阶的分频，第二个每个音阶的不同音高的分频（共有4个8度）
    控制模块中通过晶振分频获得一个地址查询时钟（8Hz），控制每首歌曲的音符（十六分之一音符），每次切换到下一个地址就把对应的分频系数传输到两个分频器中，以实现音乐播放的功能
